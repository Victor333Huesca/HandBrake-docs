Audio Mixdowns (Abmischungen)
====================================
 
Verfügbare Abmischungen (Down Mixes)
--------------------
HandBrake unterstützt die folgenden Abmischungen:
-   Mono
-   Mono (Nur Links)
-   Mono (Nur Rechts)
-   Stereo
-   Dolby Surround
-   Dobly Pro Logic II
-   5.1 Kanäle
-   6.1 Kanäle
-   7.1 Kanäle
-   7.1 Kanäle (5F/2R/LFE)


Upmixing
--------
HandBrake unterstützt derzeit kein upmixing von Audio.
